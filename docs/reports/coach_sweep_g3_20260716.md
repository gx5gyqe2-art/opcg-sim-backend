## 2. g3 フル採点（CPU=p2・34決定・worlds4/sims32 スクリーニング→16世界で精密確認）

- **勝敗断定（正味不一致≥3）に至る損失: 0 件**。スクリーニングで勝+2 だった2候補
  （@68・@75）は16世界の精密判定でどちらも正味+2＝同価値圏（@68: 11/16 vs 9/16・
  @75: 11/16 vs 9/16）。この対局の CPU に「証明可能な大失着」は無かった。
- **勝ち方の質（ライフ差 ≥0.5）の指摘: 15 件**。系統的な内容は3つ:
  1. **@93〜96 の連鎖（T10）**: 付与や過剰展開をやめ「ルフィを出してルフィで殴る」が
     ライフ +1.25〜+2.0 ——**人間マーク @93「無駄ドン付与するな」と同方向をスイープが
     独立に再発見**した。
  2. **@68/@75/@137（付与の価値）**: 余りドンの付与（ゼウス/マルコ）がライフ +1.0〜+2.25。
     方向は人間マーク（付与せよ）と一致。
  3. **序盤の攻め急ぎ（@15/@19/@43）**: T4 の無理攻めより TURN_END/別攻撃が上
     （ただし正味+1＝弱い証拠・参考）。
- **負け確定終盤（@140〜145）**: 全ドン注ぎ込み攻撃を OK/同価値と判定＝見込みの無い局面で
  作法に難癖をつけない（@93 root 全数の教訓と整合）。

## 3. @68 の経緯（訂正を含む・計器の較正が判定を変えた実例）

1. 8世界 sims64: 素攻撃 6/8 vs 付与 5/8 →「レフェリーが人間に反論」と当初報告。
2. v2 対判定では正味+1＝**同価値圏**（反論は言い過ぎだった＝訂正）。
3. 16世界 sims32: 付与→ゼウスで攻撃 12/16 vs 素 9/16（正味+3＝**断定**）・
   付与→リーダー攻撃 11/16 vs 素 9/16（正味+2＝同価値圏）。
4. 結論: **@68 は「付与が正しい」方向へ収束**（断定は「付与→ゼウスで攻撃」の形のみ）。
   人間マークの方向を支持。測定を重ねるほど人間の直感に近づいた点は、計器の
   信頼性（誤った断定をしない・証拠が集まれば断定する）の実証でもある。
