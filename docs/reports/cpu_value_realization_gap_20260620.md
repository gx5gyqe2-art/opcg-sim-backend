# CPU 精度: value-realization gap の発見と settle 楽観是正（C-5）— 2026-06-20

特定時点のスナップショット（追記・改変しない）。実ケース（ナミ対面でドン3枚を空振り）を起点に、
CPU 探索の**非定常な楽観（手番頭で過剰コミット→崩落）**を計測指標として可視化し、settle 葉の
楽観是正（C-5）を候補として実装した記録。

## 1. 実ケース（症状）

`difficulty=hard` の対局トレース（p2=CPU・リーダー OP16-080 / 対面 p1=ナミ OP11-041）。ターン3で：

1. `ATTACH_DON → バスコ・ショット(OP16-110)` ×3（付与時の採用手 deep ≈ **+4798 → +4646 → +4313**）
   - バスコ・ショットは **1 コスト・パワー2000・カウンター1000**、効果は【KO時】のみ＝**ドン!!×N 防御能力なし**。
   - 3枚付与で 2000→5000。付与は「ナミ素パワー5000 に届く」と静的層が判定（後述）。
2. `ATTACK` の候補で全選択肢が負：リーダー攻撃 **deep −91.7**（採用）／TURN_END −1028.3／
   **バスコ攻撃 −2050.7**。
3. TURN_END。→ **付与した3枚は攻撃に使われず手番終了で全返却＝完全な空振り**（守りにも使えない）。

### 根因（確定）
- **静的戦闘プルーナが相手の受け身バフを見落とす**（`_attach_don_meaningful`/`_prune_futile_attacks` は
  `get_power(False)`＝素パワーのみ）。ナミの **【相手のアタック時】手札1枚を捨て +2000**（5000→7000）を
  考慮しないため、5000 のバスコが「ナミに届く」と誤認 → 付与・攻撃とも候補に残る。実戦闘（deep）は +2000 を
  解決するので 5000<7000 で弾かれ、攻撃は −2050。
- **settle（予算切れ葉）の楽観**（主機構）。`HARD_PER_MOVE_BUDGET=150`・`HARD_HORIZON=4`・ビーム3 では
  4ターン×全展開を読み切れず、多くの葉が `_settle_eval`＝**相手のターン開始で静的採点（相手の反撃を読まない）**。
  付与の代償（相手の貫通不可／反撃）は手番頭では予算地平線の外 → deep が楽観的に +4798。攻撃の決定まで来て
  初めて予算内に入り真実（−91〜−2050）が露見。**手番ごとに探索を貼り直す**ため評価は非定常で、真実が見えた
  頃にはドンは消えている。
- **regret 指標では検知不能**。`regret = deep_best − deep_greedy` は **deep を正解とみなす**ため、deep 自身が
  楽観視するこの誤りに反応しない。

## 2. 新指標: value-realization gap（D・実装済み）

`tests/cpu_arena.py realize`（`decide_with_regret(out=…)` で採用手の深掘り値を取得）。1 ターン内で採用手の
深掘り値が `max → 最終決定` でどれだけ崩落したかを gap として (player,turn) ごとに集計：

    OPCG_LOG_SILENT=1 python tests/cpu_arena.py realize --difficulty hard --seed 0

### 計測（自己対戦・hard）
seed=0（B 無効＝現行）: 9 ターン採点で **mean_gap ≈ 6723 / p95 ≈ 17955 / max ≈ 17955 / big_gaps(≥2000)=6**。
→ 手番頭の採用手 deep が、その手番の進行中に平均 6700・最大 18000 点崩落している。崩落は特定カードに
限らず広汎（big_gaps 多数）。

## 3. 対策候補: C-5 settle 楽観是正（実装済み・**Elo 未検証**）

`_settle_eval` の打ち切り葉（相手ターン静止点・plan 供給時のみ）で、相手の次ターンの**受け切れない打点本数**
（`_incoming_reach` = リーダー＋場の素パワーが自リーダーに届く本数 − 自アクティブブロッカー）×
`W_SETTLE_PRESSURE`(=2500) を減点。**致死（reach ≥ 自ライフ）は C-2 telegraph が `evaluate` 内で計上済み
なので致死未満のみ**（二重計上回避）。`evaluate`/C-2 は不変＝**読み切れた真の地平線葉は触らず、読み切れ
なかった葉だけ**をペッシミ寄せ（C-4 と同系統）。

  候補生成段で消える（5000<7000）。C-5（汎用・機構是正）と直交。
