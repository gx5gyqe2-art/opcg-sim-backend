## 2. 判定の根拠（10kゲート・4点バッテリー）

1. **対L1多様97 = 0.854** — 事前登録基準「0.83超え=(b)成立」をクリア（写経0.771／Gen2 0.833 がバー）。
2. **マーク@47維持** — 人間の指摘（カウンター温存でゲームを伸ばす）と一致する判断を ship/写経が出せない中で唯一選択。
3. **対ship直接対戦 0.458**（48局CRN）＝教師と互角（統計的優位までは主張しない・N=24/48 はCI広）。
4. **特徴活性化プローブ** — 相手リーダー感度 ×8.8 等、v1教師が見えない特徴が実戦追い学習で活性化。

## 3. 15k 追跡測定＝「round60 がピーク」（過学習ドリフトの実証）

round93（実効15,104局）で再測定した結果、**3軸すべてが下向き**:
マーク@47 が PASS に退行（忘却）・対写経 0.375→0.312・対L1 0.854→0.792。
勝敗ラベル単独の自己対局を続けると**自己対局均衡（弱教師アトラクタ）へ回帰し、獲得した新判断が
洗い流される**。Gen3=round60 は「教師の型＋新判断」が両立した一過性の最適点であり、
凍結採用は結果的に正しかった。追い学習 run は実効17,152局で停止（ユーザ承認・2026-07-11）。

## 4. 確立した知見（次の設計への入力）

- **蒸留→実戦追い学習は休眠特徴を活性化し教師超えを起こせる**（機序: 実戦勝敗ラベルは v1教師に
  見えない v3特徴にも信号を載せる）。
- **ただし勝敗ラベル単独では過ピーク後に忘却が起きる**（15k測定）。継続学習には
  **忘却抑制（教師値ラベル×実戦勝敗の混合損失、または MCTS改善値ラベル）**が必要＝次ルートの本命。
- 対L1（αβ vs NN-MCTS）は water-oil で系統誤差あり。**同種net直接対戦（CRN）＋人間マーク回帰＋
  特徴プローブの3層評価**が「新しい考え方を獲得したか」の判定に有効だった。
- 直接対戦の非推移性: pd は ship と互角でも、同一特徴空間の出発点（写経）には相性負けする。
  1対戦相手の勝率だけで採否を決めない。

## 5. 運用インシデント（再発防止は実装済み）

- **push失敗デッドロック**: pd_gen が push 失敗でも batch_id を進め、learner の consumed が凍結→
  「learner待機」の誤診で永久停止（w2/w3 が1h停止）。修正＝配達成功まで batch_id を消費しない＋
  バックオフ付きリトライ（842d7c9）。
- **シード重複**: ワーカーIDをシードに混ぜず w1/w2 が同一ゲーム列を生成（〜実効4,100局を汚染）。
  修正＝crc32(WID) 混合（9fe9dbd）。汚染期は測定値を割引して解釈（5k中間 0.729 の停滞は汚染起因）。
- **ネット取り違え防止**: `OPCG_P3_LEAD_SLOTS`/`OPCG_P3_EFF_DIM` ガードで別アーキ checkpoint の
  サイレント学習を防止（LC事故の再発防止）。
