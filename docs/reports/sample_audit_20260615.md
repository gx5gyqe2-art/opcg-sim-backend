# サンプル精査レポート — 「✓」信頼度の実測（2026-06-15）

> 種別: 報告（点）。2026-06-15 時点のスナップショット。以後の改変はしない（履歴として残す）。
> 関連: `docs/TEST_SPEC.md` §8.4 チェックリスト／§8.6 検証計画。ハーネス: `tests/sample_audit.py`。

## 1. 目的

弾×色「✓」（§8.2 台帳）は自動ラチェット（構造監査／オラクル／挙動ベースライン／構造不変4スキャン）で
**退行**は守られているが、それらが測らない **意味的欠陥（カテゴリH 型の見落とし）** の残存率は人手の
精査でしか測れない。本レポートは、各弾からランダム抽出した標本を §8.4 で精査し、「✓」の信頼度を実測する。

## 2. 方式

- **標本**: `tests/sample_audit.py`（seed=20260615・per_set=10）で各弾から決定的にランダム抽出＝
  **456 枚 / 53 弾**（能力を持つカードのみ）。再現可能。
- **自動スクリーニング**（標本全数）: HAS_OTHER（未実装句）／STRUCT（構造不変4スキャン）／
  ORACLE（effect_oracle 高シグナル）／TEXT_ACTION_GAP（テキスト主要動詞に対応する AST アクション欠如）。
- **人手精査**: 自動フラグ付き全数（21枚）＋無フラグからの無作為抽出（28枚）を、生テキスト↔AST↔
  実行観測で §8.4（発火/条件/条件偽/対象/値/持続/複合句/コスト/安全性）に沿って判定。

## 3. 結果

### 3.1 自動スクリーニング
- HAS_OTHER / STRUCT / ORACLE: **0 枚**（既存ラチェットが全カードで 0 のため標本でも 0）。
- TEXT_ACTION_GAP: **21 枚** フラグ。**全数を精査した結果、全て誤検知**:
  - `REST_DON`（「ドン!!をレストにできる/相手のドンをレストにする」＝ドンのレストで REST ではない）
  - 過去形トリガー（「登場させた時」＝トリガー条件であってアクションではない）
  - `REPLACE_EFFECT.sub_effect` 内のアクション（「代わりに戻す/捨てる/引く」＝置換の内側。スキャナが
    意図的に sub_effect を辿らないため表層に出ない。sub_effect を確認し全て正しい実装を確認）。

### 3.2 人手精査（無フラグ無作為28枚）
- 27枚は text↔AST↔挙動が一致し健全。
- **1枚に意味的欠陥を発見＝ST23-002 シャンクス**:
  「リーダーが特徴《赤髪海賊団》を持つ**か「ウタ」**の場合」が **LEADER_TRAIT 単独**にパースされ、
  名前OR（「ウタ」）が脱落。ウタリーダーは特徴《FILM》で《赤髪海賊団》を持たないため、**ウタデッキで
  本来発火すべき登場時バフが常に不成立**だった（§8.3「条件の退化」型。自動ゲートは綺麗にパースされ
  OTHER化しないため素通り）。
  - **横展開調査**: 同型「特徴《X》を持つか「Y」の場合」は全DBで **2枚**（ST23-002／EB02-006 ヤマト
    「ワノ国 or エース」）。両方とも同じ脱落。
  - **是正**: 条件パーサで trait と「か「name」」併記時に `OR(LEADER_TRAIT, LEADER_NAME)` を生成。
    回帰 `test_leader_trait_or_name_condition`。本レポートと同 PR で修正。

## 4. 「✓」信頼度の数値化（実測）

- 精査総数 **49 枚**（自動フラグ21＋無作為28）。
- 自動ゲートが見落とした**真の意味欠陥 = 1 クラス（2枚／全DB）**。精査標本での出現は 1/49 ≒ **2%**。
- 解釈: 退行は自動ラチェットで堅く守られているが、**「✓」は“正しさ”の完全保証ではない**（カテゴリH
  ポストモーテムの教訓どおり）。人手精査は自動ゲートが構造的に測れない条件退化を確かに拾う＝本手法は
  有効。今回の1件は即時是正済みで、同型は全DB網羅して 0 に締めた。

## 5. 残課題・運用

- TEXT_ACTION_GAP ヒューリスティックは誤検知が多い（REST_DON/過去形/置換内側）。今後使うなら
  これらを除外する精緻化が必要。STRUCT/ORACLE/HAS_OTHER は信頼できる。
- 「✓」信頼度の継続的底上げには、本ハーネスを seed を変えて定期実行し、無フラグ無作為標本を人手精査
  する運用が有効（自動ゲートの死角＝意味欠陥のサンプリング監視）。
