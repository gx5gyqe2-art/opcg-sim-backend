# リーダーカード効果 仕様（全137枚）

全リーダーカードのテキストと、テキスト準拠の期待挙動・テストケースをセット別にまとめた仕様。
原文テキストが仕様の基準。各カード節は「基本（ステータス）／効果テキスト／期待挙動（テキスト準拠）／
テストケース」で構成する。

- 作成ガイド・テンプレート: [_GUIDE.md](_GUIDE.md)
- テスト方針・ヘルパAPI: [_TEST_GUIDE.md](_TEST_GUIDE.md)
- 挙動の確認補助（テキスト/パースAST/実行観測の出力）:
  `OPCG_LOG_SILENT=1 python tests/leader_spec_probe.py <ID>|--set <SET>|--all`

## 使い方

1. 個別カードの期待挙動・テストケースは各セットファイルの該当カード節を参照。
2. 挙動テストは `tests/test_leader_<set>.py`。期待挙動をアサートし、テキスト準拠の正しい挙動を固定する。
3. テキスト準拠の期待と現挙動の差異は対応するテストで表現する（テスト方針は [_TEST_GUIDE.md](_TEST_GUIDE.md)）。
