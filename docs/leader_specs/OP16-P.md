## OP16-001 ポートガス・D・エース
- **基本**: 色=赤 / ライフ=5 / パワー=5000 / 属性=特 / 特徴=白ひげ海賊団
- **効果テキスト**:
  > 【起動メイン】【ターン1回】自分のパワー8000以上の、「モンキー・Ｄ・ルフィ」か『白ひげ海賊団』を含む特徴を持つキャラ1枚までは、このターン中、【速攻】を得る。
- **期待挙動（テキスト準拠）**:
  - 能力0 【ACTIVATE_MAIN】（ターン1回）: 効果=GRANT_KEYWORD（速攻, THIS_TURN） / 対象=自分のキャラ（**パワー8000以上** かつ（名前「モンキー・D・ルフィ」 または 特徴《白ひげ海賊団》））1枚まで
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分場にパワー8000以上の《白ひげ海賊団》キャラ | 能力発動 | そのキャラが今ターン速攻獲得 |
  | 2 | 自分場にパワー7000の《白ひげ海賊団》キャラのみ | 能力発動 | 対象外（is_up_to で0枚）→付与なし |
  | 3 | パワー8000以上だが名前/特徴いずれも非該当 | 能力発動 | 対象外 |
  - 境界/異常系: パワー8000ちょうど=対象 / 7000=非対象 / ターン1回

  |---|----------|------|----------|
  | 1 | 相手ターン・自分キャラ複数 | （常在） | 自分の全キャラのコストが+1 |
  | 2 | 相手アタック宣言・手札に【トリガー】持ち | 能力1発動 | 1枚捨て→アタック対象をリーダー/《黒ひげ海賊団》キャラへ変更 |
  | 3 | 手札に【トリガー】持ちが無い | 能力1発動 | コスト払えず不可 |
  - 境界/異常系: コスト+1が相手ターン中のみ有効か / リダイレクト先が LEADER と《黒ひげ海賊団》キャラに限定 / ターン1回

## P-117 ナミ
- **基本**: 色=青 / ライフ=5 / パワー=5000 / 属性=知 / 特徴=東の海
- **効果テキスト**:
  > ルール上、自分は特徴《東の海》を持つカードしかデッキに入れることができず、自分のデッキが0枚になった場合、自分は敗北する代わりに勝利する。 / 【ドン‼×1】このリーダーのアタックによって、相手のライフにダメージを与えた時、自分のデッキの上から1枚をトラッシュに置いてもよい。
- **期待挙動（テキスト準拠）**:
  - 能力0 【PASSIVE/常在】（条件: 自分デッキ0枚）: 効果=VICTORY（敗北置換＝REPLACE_DECKOUT_LOSS）。デッキ構築制約《東の海》限定はルール文。

- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分デッキ0枚 | デッキ0到達 | 敗北せず勝利（置換） |
  | 2 | DON1付与・リーダーアタックが相手ライフにダメージ | ダメージ誘発 | デッキ上1枚をトラッシュ（任意） |
  | 3 | DON未付与 | 同上 | 発動不可 |
  - 境界/異常系: 勝利置換のタイミング / 「してもよい」任意 / アタックがブロック/カウンターで通らなかった場合は不発
