- **期待挙動（テキスト準拠）**:
  - 能力0 【アタックした時／アタックされた時】: コスト=なし / 効果=手札からイベントorステージを任意枚数捨てる(任意=してもよい) → 捨てた枚数×1000、このバトル中このリーダーにパワー+ / 対象=自分の手札(EVENT,STAGE) / 数値=捨て枚数×1000 / 期間=THIS_BATTLE / 条件=なし / ターン1回=なし
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 手札にイベント2枚 | アタック時に2枚捨て | リーダーパワー+2000(このバトル) |
  | 2 | 手札に捨てる対象なし | アタック時 | 何も捨てず+0、能力スキップ可 |
  | 3 | アタックされた時 | 同上トリガー発火 | 捨て枚数×1000 |
  - 境界/異常系: イベント/ステージ以外(キャラ)は捨て対象外 / 0枚選択も可(任意)
