## OP01-001 ロロノア・ゾロ
- **基本**: 色=赤 / ライフ=5 / パワー=5000 / 属性=斬 / 特徴=超新星・麦わらの一味
- **効果テキスト**:
  > 【ドン‼×1】【自分のターン中】自分のキャラすべてを、パワー+1000。
- **期待挙動（テキスト準拠）**:
  - 能力0 【YOUR_TURN / 常在(ドン‼×1)】: 効果=BUFF / 対象=自分のキャラ全て(ALL) / 数値=+1000 / 期間=継続(ドン‼×1が乗っている間) / 条件=自分にドン‼1枚以上付与 / ターン1回=なし
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分キャラ2枚、リーダーにドン‼1枚 | 能力発動 | 自分キャラ全てパワー+1000 |
  | 2 | 自分キャラ0枚、ドン‼1枚 | 能力発動 | 変化なし(対象不在) |
  | 3 | ドン‼0枚 | 能力発動 | 発動不可(条件未達) |
  - 境界/異常系: ドン‼を外すとバフ解除されるか / 相手キャラに乗らないこと

## OP01-002 トラファルガー・ロー
- **基本**: 色=赤/緑 / ライフ=4 / パワー=5000 / 属性=斬 / 特徴=超新星・ハートの海賊団
- **効果テキスト**:
  > 【起動メイン】【ターン1回】②(コストエリアのドン‼を指定の数レストにできる)：自分のキャラが5枚いる場合、自分のキャラ1枚を、持ち主の手札に戻し、自分の手札から、戻したキャラと異なる色のコスト5以下のキャラカード1枚までを、登場させる。
- **期待挙動（テキスト準拠）**:
  - 能力0 【ACTIVATE_MAIN / ターン1回 / コスト=ドン‼2レスト】:
    - 条件: 自分のキャラが**5枚いる場合**
    - 効果1 BOUNCE: 自分キャラ1枚を**持ち主の手札**に戻す
    - 効果2 PLAY_CARD: 自分手札から「**戻したキャラと異なる色**」かつコスト5以下のキャラ1枚**まで**を登場
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分キャラ5枚、手札にコスト5以下キャラ、ドン‼2 | 発動 | キャラ1枚戻し→異色コスト5以下を登場 |
  | 2 | 自分キャラ4枚 | 発動 | 何も起きない(条件未達) |
  | 3 | 戻したキャラと同色のキャラしか手札にない | 発動 | 登場は0枚(まで=is_up_to) |
  - 境界/異常系: コスト6以上は登場不可 / 「異なる色」制約 / ターン2回目は不可
