# リーダーカード効果 仕様 — EB-PRB



各カードのテキストと、テキスト準拠の期待挙動・テストケースを記載する。原文テキストが仕様の基準。

## EB01-001 光月おでん
- **基本**: 色=赤/緑 / ライフ=4 / パワー=5000 / 属性=斬 / 特徴=ワノ国・光月家
- **効果テキスト**:
  > ルール上、自分の特徴《ワノ国》を持ちカウンターを持たないキャラカードすべてはカウンター+1000を持つ。 / 【ドン!!×1】【アタック時】自分のコスト5以上の特徴《ワノ国》を持つキャラがいる場合、このリーダーは、次の自分のターン開始時まで、パワー+1000。
- **期待挙動（テキスト準拠）**:
  - 能力0 【PASSIVE】: ルール処理（RULE_PROCESSING）。自分の《ワノ国》かつカウンター非所持キャラすべてにカウンター+1000を付与。
  - 能力1 【ON_ATTACK】（条件: DON≥1 AND 自分場に「コスト5以上の《ワノ国》キャラ」が存在）: 効果=BUFF 自リーダー +1000 / 期間=次の自分のターン開始時まで（UNTIL_NEXT_TURN_END相当）
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | DON1付与・自分場にコスト5以上《ワノ国》キャラ有 | アタック宣言 | リーダー +1000（次自ターン開始時まで） |
  | 2 | 自分場に《ワノ国》キャラはいるがコスト5未満のみ | アタック宣言 | 不発（条件未達） |
  | 3 | 自分場の《ワノ国》キャラが《ワノ国》以外特徴のみ | アタック宣言 | 不発 |
  | 4 | 自分場に《ワノ国》カウンター非所持キャラ | 常時 | 当該キャラがカウンター+1000を獲得 |
  - 境界/異常系: コスト5境界（5は条件成立/4は不成立） / DON未付与時の不発 / 持続が「次の自分のターン開始時まで」で正しく失効するか

## EB01-021 ハンニャバル
- **基本**: 色=青/紫 / ライフ=4 / パワー=5000 / 属性=斬 / 特徴=インペルダウン
- **効果テキスト**:
  > 【自分のターン終了時】自分のコスト2以上の特徴《インペルダウン》を持つキャラ1枚を持ち主の手札に戻すことができる：ドン!!デッキからドン!!1枚までを、アクティブで追加する。
- **期待挙動（テキスト準拠）**:
  - 能力0 【TURN_END】（コスト: BOUNCE 自分のコスト2以上《インペルダウン》キャラ1枚を持ち主の手札に戻す、任意）: 効果=RAMP_DON ドンデッキからドン1枚までをアクティブで追加
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分のターン終了時・場にコスト2以上《インペルダウン》キャラ有・ドンデッキ残有 | 能力発動 | 当該キャラを持ち主手札に戻し→ドン1枚アクティブ追加 |
  | 2 | コスト2以上《インペルダウン》キャラ不在 | ターン終了 | コスト払えず不発 |
  | 3 | ドンデッキ枯渇 | 能力発動 | コスト払うがドン追加0（1枚まで=is_up_to） |
  - 境界/異常系: コスト2境界（2は対象/1は非対象） / 「持ち主の手札」へ戻る（自分のキャラなので自分手札） / アクティブで追加であること

- **期待挙動（テキスト準拠）**:
  - 能力0 【PASSIVE】（ターン1回）置換効果: 自分の「元々のコスト4以上」のキャラがKOされる場合、代わりに自分の手札1枚を捨てることができる（KO置換）
  - 能力1 【ACTIVATE_MAIN】（コスト: REST 自リーダー）: seq = 相手キャラ1枚まで THIS_TURN パワー-2000 → その後 自分の**【アタック時】効果を持たない**キャラ1枚まで THIS_TURN【速攻】付与
- **テストケース**:
  | # | 前提盤面 | 操作 | 期待結果 |
  |---|----------|------|----------|
  | 1 | 自分の元々コスト4以上キャラがKOされる・手札有 | 置換選択 | KOを防ぎ代わりに手札1枚捨て |
  | 2 | リーダーアクティブ・相手キャラ有・自分にアタック時効果なしキャラ有 | 起動メイン発動 | リーダーレスト→相手-2000→アタック時効果なしキャラに速攻付与 |
  | 3 | 自分のキャラが全て【アタック時】効果持ち | 起動メイン発動 | 速攻付与の対象なし（is_up_to 0枚） |
